# Algorithm for initial solutions

Algorithms for computing initial solutions for the CG:SHOP 2026 competition. (related to section 2)

## How to run

1. First, put the instance file(s) into `data/benchmark_instances` (if necessary).

2. Then, use the following command (instance number is order of instances in a folder) :

```bash
python main.py -n [instance number] -l [log file name]
```

3. The results will be stored in the `/solutions` folder, and the logs will be stored in the `/log` folder.
